Bienvenue sur
CopyPasta

CopyPasta est un outils qui vous permet de scanner un texte ou une image (QR Code, Code Barre, Reconnaissance de texte, ...) avec votre téléphone et d'envoyer le scan à votre ordinateur. CopyPasta vous permet aussi d'envoyer n'importe quel type de fichiers de votre téléphone vers votre ordinateur.

Scanne les QR Codes et les codes barres

Reconnaissance de textes

Reconnaissance d'objets

Envoie tous types de fichiers directement sur votre ordinateur

CopyPasta extrait les adresses mail et les URLS des scans de texte pour vous offrir une experience optimale

Comment ça marche ?

Pour utiliser cette applications, vous devez suivre ces instruction simples :

  • Téléchargez l'application mobile et le logiciel ici.
  • Ouvrez le logiciel et l'application mobile.
  • Sur votre téléphone, cliquez sur "Ajouter un appareil" et scannez le QR Code qui apparaît à l'écran de votre ordinateur.
  • C'est fini ! Vous avez juste à sélectionner votre ordinateur sur l'application mobile et vous pouvez dès à présent utiliser CopyPasta !
  • Noubliez pas de lancer CopyPasta en tant qu'Administrateur la première fois après chaque démarrage de l'ordinateur ;)
Si vous n'arrivez pas à envoyer vos scans à votre ordinateur, vérifiez bien que votre téléphone soit bien connecté sur le même réseau WiFi que votre ordinateur.
